Detailed Description

template<typename T, typename Alloc = std::allocator<T>>
class nabla::STLVectorEx< T, Alloc >

This vector is an modified version, conforming part of STL standard It must accept a pointer to allocator, and will not be responsible for destroying allocator This is NOT thread safe

Note
This vector does NOT owns allocator and it will NEVER destroy it's allocator
